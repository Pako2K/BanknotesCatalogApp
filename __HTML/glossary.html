  • Denomination:

    • Is the real value of the banknote, in the corresponding currency

  • Face Value

    • Is the value of the banknote, in the corresponding currency units given on the banknote itself

  • Issue

    • Is a semi-arbitrary grouping of one or more denominations with a common design and or similar date of issue.

    An "issue" should not include 2 banknotes with the same denomination and a different design.

    Only variations of the same denomination banknote are allowed in the same Issue.

    Exception: so called provisional/emergency issues which often include overstamped notes of different issues, which might belong to a different currency or country.
